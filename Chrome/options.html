Extension options

You need to logoff and login back into the game for the changes to be effective

  • Item Offers

    If enabled, this option will make you pay for the delivery of items, preventing undeliverable items because the other person is not in the same place as you.

  • SpeedCalling

    Enables the SpeedCalling feature, which allows you to call all your friends in your address book automatically with a single click.

    If enabled, this option will use Gossip on the phone as the default option in SpeedCalling instead Wazzup call.

  • Recipes

    If enabled, this option will add two buttons to your recipes page allowing you to fold the groups of recipes, improving readability.

  • Global Average Fame

    If enabled, this option will add to your artist's Popularity page the average global fame and popularity.

  • Suggested Ticket Prices

    If enabled, this option will show the suggested price for tickets according to your artist's fame in a given city.

  • Use higher price set for suggested Ticket Prices.

  • Quality to Value

    If enabled, will display numbers along with nominal qualities.

    High colors                                                                                                                                                                        

    Pastel colors                                                                                                                                                                        

    If enabled, will display numbers along with progress bars.

  • Filter items

    If enabled, will add an option in your locale pages to filter items by a character ID.
